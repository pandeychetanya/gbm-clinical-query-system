How to Use This System

This interface connects to the real GBM AI system running on your local computer. Follow these steps to get started:

Quick Setup (5 minutes):
  1. Clone the repository:
    git clone https://github.com/pandeychetanya/gbm-clinical-query-system.git
  2. Install dependencies:
    pip install -r requirements.txt
  3. Create the clinical database:
    python3 create_vector_db.py
  4. Start the API server:
    python3 minimal_server.py
  5. Use this interface: Once your server is running, this page will connect automatically!
